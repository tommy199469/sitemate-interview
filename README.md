# Sitemate Interview Test

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. creating a .env file and put these two param
   EXPO_PUBLIC_ARTICLE_API_ENDPOINT=https://newsapi.org/v2/everything
   EXPO_PUBLIC_ARTICLE_API_KEY=[using_your_key]

3. Start the app

   ```bash
    npm run ios
   ```
